Some Projects

Spellthief/C++ Realtime Array Parser
This was my first big programming project, and remains one of my favorites. Spellthief was designed to make players reverse engineer a programming language in order to craft their own spells and abilities. I built my own game engine in C++, and built a custom programming language into the engine. Every action of every object in the game was performed by executing a line of code in this language. The programming language, engine, and tools for level creation were fully functional, and I successfully produced and player-tested several levels.
Pathfinding robot
I built a simple robot to demonstrate a reinforcement learning AI that I created and got it to demonstrate pathfinding capabilities on limited hardware. I found ways to compensate for a low-resolution color sensor, a lack of motor encoders, and battery drainage.

The robot was built under tight budget constraints. and had to demonstrate its abilities within a time limit, which is especially limiting for a model-free reinforcement learning model. It performed adequately despite these constraints.
Messaging app web API, web client, and mobile app
I created a Web API in ASP.net designed for group messaging with as little storage space as possible. The server dynamically scales its memory usage as users connect and disconnect from the chat, and deletes messages from the server once every user has received them. I deployed this API to Azure with CORS enabled in order to consume its services from a mobile app.
Blind Bidding Game
I taught myself Xamarin by building a turn-based strategy game. The game used most major features of the Xamarin platorm, including PCLs, data binding, and a 100% shared codebase for iOS and Android using Xamarin Forms. I used the app to demonstrate my Xamarin knowledge to a potential client, and was offered a spot on the project.
Browser-based Physics Engine
I'm currently creating a physics engine using HTML, CSS, and javascript, using system modeling techniques from my background as a mechanical engineer.
